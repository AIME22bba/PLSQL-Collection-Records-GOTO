 ADD'S CUISINE — PL/SQL Demonstration Project

This repository contains PL/SQL examples created for the **ADD'S CUISINE Smart Food Delivery System**.  
The project demonstrates the use of:

PL/SQL Collections**
  PL/SQL Record Types**
  PL/SQL GOTO Statements**

These examples are part of an academic exercise to understand how PL/SQL can be used to manage order data, validate inputs, and store multiple values efficiently.
